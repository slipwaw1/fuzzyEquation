# FuzzyEquation

Інформаційна система для генерування нечітких логічних рівнянь.

## Загальна характеристика

FuzzyEquation — це веборієнтована інформаційна система, призначена для автоматизованого формування нечітких логічних рівнянь на основі заданих користувачем критеріїв оцінювання та відповідних лінгвістичних термів.

Система реалізована як автономний клієнтський застосунок, що функціонує у середовищі веббраузера без необхідності встановлення додаткового програмного забезпечення. Такий підхід забезпечує простоту використання, доступність та незалежність від операційної системи.

## Функціональні можливості

Система забезпечує повний цикл побудови нечіткої бази знань та генерування нечітких логічних рівнянь, зокрема:

- визначення кількості критеріїв оцінювання
- введення змістових назв критеріїв
- задання множини лінгвістичних термів для кожного критерію
- автоматичне формування структури матриці знань
- заповнення матриці знань шляхом вибору рівнів якості
- перевірка коректності введених даних
- генерація нечітких логічних рівнянь у символьній формі
- можливість повторного запуску процесу (скидання даних)

## Принцип роботи системи

Робота системи організована у вигляді послідовного триетапного процесу.

### Етап 1. Введення кількості критеріїв

Користувач задає кількість критеріїв оцінювання, що братимуть участь у формуванні нечітких рівнянь. Після підтвердження система динамічно формує відповідну кількість полів для подальшого введення даних.

### Етап 2. Введення параметрів критеріїв

На цьому етапі користувач вводить:
- назви критеріїв
- перелік лінгвістичних термів для кожного критерію

Терми задаються у текстовому форматі. Система виконує перевірку заповнення обов’язкових полів та запобігає переходу до наступного етапу у разі некоректних або неповних даних.

### Етап 3. Формування матриці знань

Система автоматично генерує матрицю знань у табличному вигляді:
- рядки — комбінації лінгвістичних термів
- стовпці — критерії

Для кожної комірки передбачено вибір рівня якості, що дозволяє сформувати структуру нечіткої бази знань.

Після заповнення матриці активується процес генерації нечітких логічних рівнянь.

## Генерування нечітких логічних рівнянь

У результаті роботи системи формуються нечіткі логічні рівняння у символьній формі.

У виразах використовуються стандартні позначення нечіткої логіки:
- кон’юнкція (∧)
- диз’юнкція (∨)
- функція належності (μ)

Згенеровані рівняння можуть бути використані для подальшого аналізу або побудови моделей у системах нечіткого виводу.

## Архітектура системи

Система реалізована за клієнтською архітектурою та складається з таких компонентів:

- **HTML5** — структура інтерфейсу
- **CSS3** — стилізація та адаптивний дизайн
- **JavaScript** — реалізація логіки роботи системи

Динамічне формування інтерфейсу здійснюється шляхом маніпуляції DOM-деревом документа.

Інтерфейс побудований із використанням сучасних підходів адаптивної верстки:
- CSS Grid
- Flexbox

Це забезпечує коректне відображення системи на різних пристроях.
